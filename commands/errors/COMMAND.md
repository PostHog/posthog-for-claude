# /posthog:errors

View and analyze errors tracked in PostHog.

## Instructions

Use the PostHog MCP tools to help the user with error tracking:

1. To list recent errors, use `posthog_list_errors`
2. To get details about a specific error, use `posthog_get_error_details` with the error ID
3. Help the user understand error patterns, frequency, and affected users

## Example prompts

- "What are my most common errors?"
- "Show me errors from the last 24 hours"
- "Tell me more about this error: [error_id]"
- "Which errors are affecting the most users?"
